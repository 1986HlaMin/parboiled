This example lives in the package "org.parboiled.examples.abc":http://github.com/sirthias/parboiled/tree/master/src/examples/java/org/parboiled/examples/abc/
and is a very simple demo of the classic non-context free language example { a^n b^n c^n : n >= 1 }:

bq. S ← &(A c) a+ B !(a|b|c)
A ← a A? b
B ← b B? c

See the [[Java Examples]] page for instructions on how to easily run this example on your machine.
The grammar is a very straightforward _parboiled for Java_ representation of the grammar shown above. It uses both
syntactic predicates available in Parsing Expression Grammars.
Since no actions are used this example is a very simple recognizer for the above language, nothing more.
