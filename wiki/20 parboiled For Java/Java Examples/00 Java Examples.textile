The package "org.parboiled.examples":http://github.com/sirthias/parboiled/tree/master/src/examples/java/org/parboiled/examples/ contains sub packages for each _parboiled for Java_ example. Each example contains a _Main_ class with a _public static main_ method entry point.

You can run an example by following these steps:
# Clone the source repository (either from the master branch or a specific version tag)
# Run @ant -p@ to see all the defined ant targets
# Choose one of the *run-example-XYZ* targets and run @ant run-example-XYZ@

Currently _parboiled_ has at least these examples for you to look at:
* [[ABC Grammar]]
* [[Calculators]]
* [[Java Parser]]
* [[Time Parser]]
